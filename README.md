# TSLA 5-Year Stock Analysis & Predictive Modeling

An end-to-end data science and machine learning project analyzing Tesla (TSLA) stock data over the past five years. This project combines exploratory data analysis, feature engineering, and predictive modeling to uncover trends and forecast stock closing prices.

**Key Highlights:**
- Explored long-term price trends, volatility, and trading volume dynamics  
- Engineered technical indicators such as moving averages and momentum features  
- Built and evaluated a Random Forest regression model for stock price prediction  
- Achieved robust performance: RMSE 21.46, RÂ² 0.94  
- Visualized key insights with clear, interpretable plots  

**Tech Stack:** Python, Pandas, NumPy, Matplotlib, Seaborn, scikit-learn  

**Project Artifacts:** Notebook, visualizations, and trained model included
